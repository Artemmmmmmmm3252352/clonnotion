# Anima Project

## Overview
A React project automatically generated by Anima using the Shadcn UI library. This is a task management application with project planning and task management sections.

## Tech Stack
- React 18
- Vite 6
- TypeScript
- Tailwind CSS
- Shadcn UI components (Radix UI primitives)
- React Router DOM

## Project Structure
```
src/
  components/ui/     - Reusable UI components (badge, button, card, checkbox, input, select)
  lib/               - Utility functions
  screens/           - Page components
    FigmaDesignPhoto/
      sections/      - Page sections (ProjectPlanSection, TaskManagementSection)
  index.tsx          - Application entry point
public/              - Static assets (images, backgrounds)
```

## Development
- Dev server: `npm run dev` (runs on port 5000)
- Build: `npm run build` (outputs to dist/)

## Deployment
Configured for static deployment with:
- Build command: `npm run build`
- Public directory: `dist`
